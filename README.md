# TSP Heuristics
Implementation of heuristics for the Traveling Salesman problem (TSP) in python

### Dependencies
	* mpmath

Using ```pip```

```bash
sudo pip install mpmath
```

#### Constructive Heuristics
	* Nearest Neighbour
	* Nearthest Insertion

#### Lower Bound
	* One Tree
	* Held Karp Bound

#### Execution
```bash
python main.py
```
